# Dynamic Li.Fi Gasless Swaps Demo

A demonstration of gasless token swaps using Dynamic SDK, ZeroDev, and Li.Fi.

## Installation

```bash
npm install
```

## Environment Variables

Create a `.env.local` file:

```bash
NEXT_PUBLIC_DYNAMIC_ENVIRONMENT_ID=your_dynamic_environment_id
```

## Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

- Embedded wallet authentication
- Gas-sponsored transactions via ZeroDev + Pimlico
- Token swaps through Li.Fi SDK
- No Wagmi required
